# Xelans Markets - Premium Trading Platform

A high-end, modern, and responsive landing page for Xelans Markets, featuring a sophisticated dark/light mode toggle, dynamic animations, and a premium financial aesthetic.

## 🚀 Features
- **Modern UI**: Sleek dark navy and gold design palette.
- **Dynamic Interactions**: Hover effects, smooth transitions, and premium animations.
- **Fully Responsive**: Optimized for all devices and screen sizes.
- **Glassmorphism**: Elegant card designs with glass-like effects.
- **Multi-Theme Support**: Seamless switching between light and dark modes.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure.
- **Vanilla CSS**: Advanced layout techniques including Flexbox, Grid, and Animations.
- **JavaScript**: Interactive elements and theme management.

## 📦 Local Development
1. Clone the repository.
2. Open `index.html` in your browser.
